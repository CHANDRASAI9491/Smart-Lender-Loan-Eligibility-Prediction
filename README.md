# 🏦 Smart Lender - Loan Eligibility Prediction System

## 📌 Project Overview

Smart Lender is a Machine Learning based web application that predicts whether a loan applicant is eligible for loan approval. The project uses data preprocessing, exploratory data analysis, machine learning algorithms, and a Flask web application to provide real-time predictions.

---

## 🚀 Features

- Loan Eligibility Prediction
- Data Visualization (EDA)
- Data Preprocessing
- Multiple Machine Learning Models
- Random Forest Model Deployment
- Flask Web Application
- Responsive User Interface

## ▶️ How to Run

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the Flask application

```
cd flask
python app.py
```

3. Open your browser

```
http://127.0.0.1:5000
```
